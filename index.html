CTJ Madrichim

This internal document management system is designed for a teaching assistant program at Congregation Tikvat Jacob Beth Torah. The program has thirty teenagers ranging from eighth to twelfth grades and aims to follow their development over their years of participation. The system manages their initial applications, administrator interviews, incident reports, mid-year evaluations (both for themselves and from their assigned teacher), application for the following year, year end evaluations from teachers and their exit evaluation. This allows the administrators to better respond to concerns from both teachers and teenagers.

This site was built with Ruby on Rails, self-built authentication, and mongoDB.

Track Yo' Gas

Being able to track the miles per gallon that a car gets can help a driver drive more efficiently. This application allows users to enter tanks on different cars and follow up with how their miles per hour change. Future iterations of the project will allow users to add comments such as if the tank was meant for a long distance drive versus being stuck in city traffic and weather conditions.

This site was built with Ruby on Rails, D3, postgresql, and self-built authentication.

ParDIY

A refactor of this project is in progress. This application creates a community for users who like to plan their own parties. At the moment, there are a lot of blogs or Pinterest boards that can be used as party templates but there is not one place that a user can find all of the instructions for decorations, themed foods and other relevant ideas. Users can post a new template and future iterations are planned to allow users to sell templates and also coordinate with locations.

This site will be built with mongoDB, Devise authentication, Facebook authentication and Ruby on Rails.
